Module: xml_utils

XML utilities for TYTX Protocol.

Provides functions for XML serialization with typed values.

Structure convention: Each XML element maps to: {tag: {attrs: {...}, value: ...}} - attrs: object of attributes (can be typed with ::type) - value: scalar (string or typed), object of children, or array (repeated elements)

Usage: // Typed XML (TYTX format - reversible) as_typed_xml(data) // → '99.50::D' from_xml(xml_str) // → {price: {attrs: {}, value: 99.5}}

// Standard XML (for external systems)
as_xml(data)  // → '<price>99.50</price>'
