Module: json_utils

JSON utilities for TYTX Protocol.

JSON-native types (number, boolean, string, null) are NOT marked with type codes because JSON already preserves their type. Only non-native types (Date) receive type markers.

Usage: // Typed JSON (TYTX format - reversible) as_typed_json({count: 5, date: new Date()}) // → '{"count": 5, "date": "2025-01-15T00:00:00::DH"}'

from_json(json_str)  // → {count: 5, date: Date}

// Standard JSON (for external systems)
as_json(data)  // → '{"count": 5, "date": "2025-01-15T00:00:00.000Z"}'
