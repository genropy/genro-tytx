Base class for TYTX-aware models.

Extend this class to create models that automatically handle TYTX serialization and deserialization.

toTytx() → {string}

Serialize this instance to TYTX JSON string.

All Date objects, numbers, booleans are encoded with type suffixes.

Source:
  • tytx_model.js, line 50
Returns:

TYTX JSON string.

Type
string
Example
const order = new Order();
order.price = 99.99;
order.date = new Date('2025-01-15');
order.toTytx();
// '{"price":"99.99::R","date":"2025-01-15::D"}'
